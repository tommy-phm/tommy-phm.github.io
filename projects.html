Projects
Pool

Pool is a realistic pool game simulator. The physics engine can simulate elastic collisions and friction. Features include online coplay, cross-platform support, API, and AI players.

Chat

Chat is an online messaging application. Account information and messages are stored in a MySQL database. It supports public/private channels, 2FA email verification, and light/dark mode themes.

Uno

Uno is implementation of the game Uno. It offers online multiplayer (4 players), custome username, custom port, multithreading (for game logic and network communication).

GoL

Conway's Game of Life is a simulation of cellular automata. Customization option includes: resolution, color, sprite size, edge type (hedge, torus, klevin bottle), and file pattern.
