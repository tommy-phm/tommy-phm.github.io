Projects
Pool

Pool is a realistic pool game simulator. The physics engine can simulate elastic collisions and friction. Features include online coplay, cross-platform support, API, and AI players.

Chat

Chat is an online messaging application. It stores data in MySQL a database and supports public/private channels, 2FA email verification, and light/dark mode themes.

Uno

Uno is implementation of the game Uno. It offers online multiplayer (4 players), custome username, custom port, multithreading (for game logic and network communication).

GoL

Conway's Game of Life is a simulation of cellular automata. Customization options include: resolution, color, sprite size, edge type (hedge, torus, Klein bottle), and file patterns.
